# Маршрутные карты MES (SPA)

Статическое SPA с локальным сохранением состояния (localStorage), пригодное для публикации на GitHub Pages — достаточно раздавать содержимое папки `public/`.

## Запуск локально

Откройте `public/index.html` в браузере или поднимите любой простой статический сервер в каталоге `public`:

```sh
npx serve public
```

Данные (МК, задержки, изолятор и т.д.) хранятся в `localStorage` браузера.
