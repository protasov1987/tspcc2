# Маршрутные карты MES (SPA)

Статическое SPA с локальным сохранением состояния (localStorage), пригодное для публикации на GitHub Pages — достаточно раздавать содержимое корня репозитория.

## Запуск локально

Откройте `index.html` в браузере или поднимите любой простой статический сервер в корневом каталоге:

```sh
npx serve .
```

Данные (МК, задержки, изолятор и т.д.) хранятся в `localStorage` браузера.
